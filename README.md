## ✏️ كيفية التعديل

### تغيير اسم المكتب
ابحث عن `النخبة للمحاماة` في الملف واستبدلها بالاسم المطلوب.

### تغيير الألوان
في أول الملف داخل `<style>` ابحث عن:
```css
:root {
  --gold: #C9A84C;      /* اللون الذهبي الرئيسي */
  --gold-light: #E8C97A;
  --gold-dark: #8B6914;
  --dark: #0A0A0F;      /* لون الخلفية */
}
```

### تغيير بيانات التواصل
ابحث عن قسم `<!-- CONTACT -->` وغيّر:
- العنوان
- رقم الهاتف
- البريد الإلكتروني
- ساعات العمل

### تغيير الإحصائيات
ابحث عن قسم `hero-stats` وعدّل الأرقام:
```html
<span class="stat-num">+2000</span>  <!-- عدد القضايا -->
<span class="stat-num">20+</span>    <!-- سنوات الخبرة -->
<span class="stat-num">98%</span>    <!-- نسبة الرضا -->
<span class="stat-num">15</span>     <!-- عدد المحامين -->
```
